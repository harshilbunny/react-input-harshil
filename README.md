# react-input-harshil

## Install

```bash
npm install --save react-input-harshil
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-input-harshil'
import 'react-input-harshil/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
